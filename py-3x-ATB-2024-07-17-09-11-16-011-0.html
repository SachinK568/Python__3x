        			<p><strong>Learning Resources...</strong><br />
        Published DataFlex powered Web Services are available to assist  developers with learning and testing Web Services in  their applications. Learn more about creating <a href="https://docs.dataaccess.com/dataflexhelp/#t=mergedProjects%2FDevelopmentGuide%2Fweb_services_in_dataflex.htm" title="Web Services in DataFlex">Web Services</a> in the DataFlex Online Help.</p>
        <p>The "<a title="null" target="_blank" href="http://www.dataaccess.com/webservicesserver/numberconversion.wso">Conversions</a>" DataFlex Web Service, developed by DAW&#0146;s Marcia Booth, provides a  variety of functions that will convert numbers into words (English),  numbers into dollar amounts, and texts into title-cased texts.</p>
        <p>The service can be tried by anyone by visiting the Conversions DataFlex Web Service page hosted by Data Access Worldwide.</p>
        <p>The Conversions service has already been found useful by one enthusiastic VB.net developer:</p>
        <blockquote style="margin-right: 0px" dir="ltr">
        <p><em>I wanted to thank you for providing an extraordinary web service  by way of your Conversions WSDL! I&#0146;ve consumed this service in a  practice smart client app and was utterly amazed at how it all works! If  I had to write the code to convert numbers to words it would take a  considerable amount of time. I was able to set up a few lines of code  and consume your conversions service in LESS than 5 minutes! It&#0146;s just a  practice app but your WSDL is great! Your web service was the first one  I&#0146;ve consumed programmatically! Thanks!</em> <br />
        <br />
        Brice R. <br />
        VB.Net WinForms Developer</p>
        </blockquote>
        <p>The "<a target="_blank" title="null" href="http://webservices.oorsprong.org/websamples.anagram/anagramservice.wso">Anagram</a>" DataFlex Web Service, developed by DAW&#0146;s Vincent Oorsprong,  demonstrates functions to test your knowledge about anagrams. There are  functions to test if a word is an anagram, how many words are stored  etc. Anagrams can be stored in multiple languages.</p>
        <p>Other available DataFlex Web Services include:</p>
        <ul>
            <li><a target="_blank" title="null" href="http://webservices.oorsprong.org/websamples.arendsoog/ArendsoogbooksService.wso">Arendsoog</a> - This webservice contains information about the a Dutch book series called Arendsoog. The 64 books were published between 1934 and 1995. The books are mainly written in Dutch, the titles and detail information will be in Dutch.</li>
            <li><a target="_blank" title="null" href="http://www.oorsprong.org/websamples.countryinfo/CountryInfoService.wso">Country information</a> - This DataFlex Web Service opens up country information. 2 letter ISO codes are used for Country code. There are functions to retrieve the used Currency, Language, Capital City, Continent and Telephone code.</li>
            <li><a target="_blank" title="null" href="http://webservices.daehosting.com/services/datesservice.wso">Dates</a> - DataFlex Web Service for All kind of Date Features such as Daynames, MonthNames etc.</li>
            <li><a target="_blank" title="null" href="http://webservices.daehosting.com/services/ElevenTest.wso">Elfproef</a> - DataFlex Web Service for Checksums based on the &#0146;ElfProef&#0146;. The &#0146;ElfProef&#0146; is a CRC technique determine if a number is a valid number where the sum of the passed digits divided by 11 must be a integer. Security Note: No information about the passed numbers will be stored, we only count the number of times anyone used one of the functions of this webservice for statistic reports.</li>
            <li><a target="_blank" title="null" href="http://webservices.daehosting.com/errorinfosystem/errorinfoservice.wso">Error Information</a> - This DataFlex WebService can be used to get information about an error that can occur in a system. The errors are not DataFlex specific.</li>
            <li><a target="_blank" title="null" href="http://webservices.daehosting.com/services/ISBNService.wso">ISBN</a> - DataFlex Web Service to validate ISBN numbers.</li>
            <li><a target="_blank" title="null" href="http://webservices.daehosting.com/services/TemperatureConversions.wso">Temperature</a> - DataFlex Web Service to convert temperature values between Celsius and Fahrenheit. Can also return the windchill temperature.</li>
        </ul>
        <p>Building client applications to consume Web Services with DataFlex is fast and easy! You just need to run the DataFlex  Studio&#0146;s &#0147;Web Service Client Class Generator&#0148; to have a package  automatically created. After the package is part of their class list,  you can add the service to any of your DataFlex applications.</p>
        <p>All business solution developers are welcome to try these Web  Services. Note that any developer &#0150; not just DataFlex developers &#0150;  can use these DataFlex Web Services in their applications.</p>
        <p>Developers interested in having DAW publish a DataFlex Web  Service that they feel would be beneficial to business solutions  developers are encouraged to contact <a title="null" href="mailto:marcia.booth@dataaccess.com?subject=Publishing Web Services">Marcia Booth</a> at Data Access in Miami.</p>
